# TetrisJS

A minimal Tetris engine written in functional javascript.

Includes a CLI to play the game:

![](https://github.com/ribaptista/tetramino/blob/master/demo.gif?raw=true)

## Getting Started

### Prerequisites

```
node
```

### Installing

```
npm i
```

### Running

```
npm start
```
